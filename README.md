# mAngkot
Repository untuk aplikasi RPL Semester 3
Aplikasi m-Angkot merupakan sebuah aplikasi berbasis lokasi untuk membantu orang yang kurang mengetahui tentang rute-rute angkot di suatu daerah dalam menemukan rute angkot terbaik untuk mencapai tujuan. Aplikasi ini memiliki tiga fitur utama, yaitu Recommend Route, Search Route, dan Get Direction. Terdapat perbedaan desain prototype dan hasil final aplikasi m-Angkot yang disebabkan oleh keterbatasan waktu. Namun, secara keseluruhan, aplikasi ini telah memiliki fitur sesuai seperti yang direncanakan meski dalam desain yang sedikit berbeda.
